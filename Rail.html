Rail Fare Calculator

About The Program
This is a simple python script which was written to calculate the cheapest route between two points on a map. in this case it was a few train stations. I wanted to use dykstra's shortest path algorithm to calculate the shortest / cheapest path between two uk rail stations as i was doing a fair amount of travleing to and from placment interviews all over the country.
How it Works
The Basic idea is that it uses dykstra's algorithm to find the shortest path between two points on a graph (map) and as i said its fairly basic but it uses python as the language as this was one of the only programing langugaes i knew at this time. Dykstra's algorithm starts by starting at the last point or the destination in this case, it sets the distance or the price to 0, then explores all connected nodes and marks the shortest distance, it repeats this until the start node is found and the shortest path between these two nodes is recorded. As you can see from the pictures, the cheapest path between Nottingham and London is via Grantham even though Nottingham has a direct link to London.
