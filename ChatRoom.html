About The Program
This was written in python, much like many of the programs i made at this time, this one uses sockets to allow users to connect to a server, which acts as a chat room.
How it Works

This program was fairly straight forward to write, it accepts client connections and then allows them to send messages to the server for the other users to see. there really isnt much to this program but i thought it was worth noting because it was where i first learnt how to use sockets to host a server.
